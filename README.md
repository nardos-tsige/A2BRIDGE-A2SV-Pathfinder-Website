**Tech Stack**
  1. TypeScript — Core logic and type safety

  2. JavaScript — Interactive features and utilities

  3. Node.js — Backend runtime

  4. Vercel — Deployment

**Run Locally**

# Install dependencies
npm install

# Run the development server
npm run dev
